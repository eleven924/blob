# LangChain 基础

## 什么是 LangChain？
LangChain 是一个开源框架，用于开发由大型语言模型（LLM）驱动的应用程序。它提供了一套工具、组件和接口，帮助开发者将 LLM 与外部数据源、API 和其他服务集成，从而构建更复杂、更实用的 AI 应用。

## LangChain 与 Spring AI 对比

| 特性 | LangChain | Spring AI |
|------|-----------|-----------|
| **定位** | 通用 LLM 应用框架 | Spring 生态系统中的 AI 集成框架 |
| **主要功能** | 链（Chains）、代理（Agents）、检索增强生成（RAG） | 统一的 AI 模型 API、向量存储集成、RAG 支持 |
| **生态系统** | 独立生态，支持多种 LLM 和工具 | 与 Spring Boot、Spring Cloud 等深度集成 |
| **灵活性** | 高度灵活，支持自定义组件 | 遵循 Spring 约定，配置驱动 |
| **适用场景** | 快速开发原型、复杂 LLM 应用 | 企业级应用、已有 Spring 项目集成 AI |

## 总结
LangChain 是一个强大的 LLM 应用开发框架，适合快速构建原型和复杂的 AI 应用。而 Spring AI 则更适合企业级应用，尤其是那些已经使用 Spring 生态系统的项目。开发者可以根据自己的需求和项目背景选择合适的框架。
